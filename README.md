| **特征类别** | **特征名称** | **作用** | **计算公式** |
|--------------|--------------|----------|--------------|
| **能量特征** | 均方根值 (RMS) | 反映信号的整体能量水平，适用于评估振动强度。故障通常会导致振动能量增加，RMS值增大能够帮助识别设备故障，如轴承损坏。 | \[ \text{RMS} = \sqrt{\frac{1}{N} \sum_{i=1}^{N} x_i^2} \] |
|              | 峰值 (Peak Value) | 反映信号中出现的最大瞬时值。用于检测冲击事件，因为故障引发的冲击会使峰值增大。 | \[ \text{Peak Value} = \max(|x_i|) \] |
|              | 峰峰值 (Peak-to-Peak Value) | 表示信号中最大峰值与最小峰值之间的差异。用于评估信号的波动性，故障导致振动幅度增加，使得峰峰值增大。 | \[ \text{Peak-to-Peak Value} = \max(x_i) - \min(x_i) \] |
|              | 功率谱密度 (PSD) | 描述信号在频域中的能量分布。适用于分析随机振动和识别异常频率成分，从而发现机械故障。 | \[ \text{PSD}(f) = \frac{|X(f)|^2}{\Delta f} \] |
| **冲击特征** | 峭度 (Kurtosis) | 反映信号中峰值的尖锐程度。轴承或齿轮故障会引起冲击，导致峭度值增大。 | \[ \text{Kurtosis} = \frac{1}{N} \sum_{i=1}^{N} \left( \frac{x_i - \mu}{\sigma} \right)^4 \] |
|              | 冲击因子 (Crest Factor) | 表示信号峰值与RMS值的比值。高冲击因子表明存在突发性的冲击事件，可能是故障引起。 | \[ \text{Crest Factor} = \frac{\text{Peak Value}}{\text{RMS}} \] |
|              | 峰值因子 (Peak Factor) | 表示信号中最大的峰值幅度。高峰值因子可能反映突发冲击，通常与故障相关。 | \[ \text{Peak Factor} = \frac{\text{Peak Value}}{\text{Mean Value}} \] |
|              | 冲击脉冲指数 (Impact Pulse Index) | 反映信号中冲击脉冲的频率和强度。用于检测反复冲击事件，如齿轮缺陷。 | 通过时间域分析冲击脉冲的频率和幅值计算。 |
| **调制特征** | 幅度调制 (AM) | 反映振动信号的幅值随时间变化，通常由机械松动或齿轮故障引起。通过包络变化可以识别这些故障。 | 包络检测算法获得包络信号，分析其变化。 |
|              | 频率调制 (FM) | 反映振动信号的瞬时频率变化。通常由转速波动或传动系统故障引起。 | 通过希尔伯特变换计算瞬时频率。 |
|              | 相位调制 (PM) | 反映信号相位随时间变化，通常由周期性机械故障引起。 | 计算信号相位并分析其变化。 |
|              | 调制谱 (Modulation Spectrum) | 通过对调制信号进行傅里叶变换得到频谱，显示调制成分的频率和幅值。用于检测周期性变化。 | 对包络信号或瞬时频率进行傅里叶变换。 |
| **谐波特征** | 谐波含量 (Harmonic Content) | 识别基频及其谐波成分的幅值和频率。用于检测非线性效应或周期性故障。 | 通过傅里叶变换获得频谱，识别并量化基频及其谐波成分的幅值。 |
|              | 总谐波失真 (THD) | 反映谐波成分相对于基频成分的总能量。用于评估信号的谐波失真程度。 | \[ \text{THD} = \sqrt{\sum_{n=2}^{\infty} \left( \frac{V_n}{V_1} \right)^2} \] |
|              | 谐波频率 (Harmonic Frequencies) | 识别振动信号中的谐波频率，常见于周期性机械运动。 | 通过傅里叶变换识别频谱中的谐波频率。 |
|              | 谐波包络 (Harmonic Envelope) | 反映谐波信号的包络变化，常见于周期性故障。 | 通过包络检测算法分析谐波信号的包络。 |
| **声品质特征** | 响度 (Loudness) | 反映声音的强度感知，是评价噪声的重要指标。用于评估机械设备运行的声品质。 | 通过心理声学模型计算，如ISO 532 B。 |
|              | 粗糙度 (Roughness) | 描述声音的刺耳程度，与调制频率和深度有关。用于检测设备噪声中的刺耳成分。 | 依赖于心理声学参数和振动信号的调制特征。 |
|              | 尖锐度 (Sharpness) | 反映声音的高频成分感知程度。用于评估高频噪声成分。 | 通过高频成分的能量计算。 |
|              | 声音品质因子 (Sound Quality Index) | 综合反映声音的总体品质，结合响度、粗糙度、尖锐度等特征进行综合评估。 | 结合各个声品质特征进行综合评估。 |
| **频率分布特征** | 中心频率 (Center Frequency) | 反映信号频谱的主要能量集中位置，帮助定位主要振动源。 | \[ f_c = \frac{\int_{0}^{\infty} f S(f) df}{\int_{0}^{\infty} S(f) df} \] |
|              | 频谱带宽 (Bandwidth) | 描述信号频谱的宽度，反映信号包含的频率范围。用于评估振动信号的频谱特性。 | 通过半功率点计算频谱带宽。 |
|              | 频谱峰值 (Spectral Peaks) | 识别频谱中显著的峰值及其频率位置。帮助检测特定故障频率。 | 通过傅里叶变换得到频谱后识别峰值点的位置和幅值。 |
|              | 频率重心 (Frequency Centroid) | 反映频谱的重心位置，衡量频率分布的中心点。用于评估频谱分布的集中程度。 | \[ \text{Frequency Centroid} = \frac{\sum_{i=1}^{N} f_i \cdot S(f_i)}{\sum_{i=1}^{N} S(f_i)} \] |
