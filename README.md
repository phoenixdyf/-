## 能量特征 (Energy Features)

### 1. RMS (均方根值)
**作用**：RMS值反映了信号的整体能量水平，适用于评估振动强度。它是评估振动信号的重要指标，因为它综合考虑了信号的所有瞬时值，使其对整个信号的能量有一个全面的评价。RMS值常用于机械设备的故障诊断，如轴承和齿轮的健康状态监测。设备运行时，若其振动信号的RMS值显著高于正常值，可能预示着某种机械故障的存在。这是因为故障通常会导致振动能量的增加，从而反映在RMS值上。因此，通过定期监测设备的RMS值，可以及早发现潜在的机械问题，避免故障的扩大和设备的损坏。

**计算公式**：

$$
\text{RMS} = \sqrt{\frac{1}{N} \sum_{i=1}^{N} x_i^2}
$$


其中，\(x_i\)为振动信号的样本值，\(N\)为样本数量。

### 2. 峰值 (Peak Value)
**作用**：峰值是信号中出现的最大瞬时值，反映了信号中最强的冲击事件。它在振动分析中具有重要意义，尤其是在检测机械冲击和瞬态事件方面。机械设备的故障，如轴承或齿轮的破损，通常会导致瞬时的高幅值振动，这些振动会显著增加峰值。因此，峰值是评估机械健康状态的关键指标之一。通过监测峰值，可以快速识别出设备的异常状态，尤其是那些可能导致严重损坏的瞬时冲击事件。相比于RMS值，峰值更加敏感，可以在早期检测到潜在的故障，从而进行及时维护和修理，防止故障进一步发展。

**计算公式**：

\[ \text{Peak Value} = \max(|x_i|) \]

其中，\(x_i\)为振动信号的样本值。

### 3. 峰峰值 (Peak-to-Peak)
**作用**：峰峰值表示信号中最大峰值与最小峰值之间的差异，是评估信号波动性的重要指标。它反映了信号的总体振幅范围，对于机械故障诊断有着重要意义。机械设备的故障，如松动、磨损或不平衡，通常会导致振动信号的波动性增加，表现为更大的峰峰值。因此，通过监测峰峰值，可以识别出机械系统中的异常波动，进而判断设备的健康状态。与峰值相比，峰峰值提供了信号振动的全貌，更全面地反映了信号的波动特性。定期监测设备的峰峰值，可以帮助早期发现潜在的机械问题，及时采取维护措施，避免故障的进一步扩展。

**计算公式**：

\[ \text{Peak-to-Peak Value} = \max(x_i) - \min(x_i) \]

其中，\(x_i\)为振动信号的样本值。

### 4. 功率谱密度 (PSD, Power Spectral Density)
**作用**：功率谱密度（PSD）描述了信号在频域中的能量分布，是分析随机振动和识别特定频率能量集中的重要工具。通过PSD，可以发现信号中的周期性成分和频率特征，帮助识别机械系统中的异常频率成分。例如，轴承或齿轮的故障会在特定频率下产生显著的振动能量，PSD能够将这些频率特征可视化，便于工程师进行故障诊断和分析。此外，PSD在噪声分析、系统特性评估和结构健康监测中也有广泛应用。通过对PSD的分析，可以获得信号的频谱特性，评估系统的动态性能，从而提高机械设备的可靠性和安全性。

**计算公式**：

\[ \text{PSD}(f) = \frac{|X(f)|^2}{\Delta f} \]

其中，\(X(f)\)为信号的傅里叶变换，\(\Delta f\)为频率分辨率。

## 冲击特征 (Impact Features)

### 1. 峭度 (Kurtosis)
**作用**：峭度反映信号中峰值的尖锐程度，是衡量信号脉冲特性的重要参数。它用于评估信号中高峰值事件的频率和强度。机械系统中的冲击性故障，如轴承或齿轮的损坏，通常会导致振动信号的峭度显著增加。这是因为这些故障会产生剧烈的冲击事件，导致信号中出现更尖锐的峰值。峭度是故障诊断中的敏感指标之一，通过监测峭度的变化，可以早期检测出机械故障。高峭度值表明信号中存在强烈的冲击性成分，可能预示着设备的健康状况不佳。相比于其他统计特征，峭度更能反映出信号中异常冲击事件的存在和严重程度。

**计算公式**：

\[ \text{Kurtosis} = \frac{1}{N} \sum_{i=1}^{N} \left( \frac{x_i - \mu}{\sigma} \right)^4 \]

其中，\(\mu\)为均值，\(\sigma\)为标准差。

### 2. 冲击因子 (Crest Factor)
**作用**：冲击因子是信号峰值与RMS值的比值，用于评估信号中的冲击成分。它在机械故障诊断中具有重要应用，特别是用于识别突发性的冲击事件。机械设备在运行过程中，若发生故障，如轴承磨损或齿轮断裂，通常会产生突发的高幅值振动，导致冲击因子显著增加。因此，通过监测冲击因子，可以有效识别设备的异常状态。高冲击因子值表明信号中存在明显的冲击成分，预示着可能的机械故障。相比于单独的峰值或RMS值，冲击因子提供了一种综合的评价方法，更敏感地反映出信号中的突发冲击事件。

**计算公式**：

\[ \text{Crest Factor} = \frac{\text{Peak Value}}{\text{RMS}} \]

该值越高，表示信号中存在更多的冲击成分。

### 3. 峰值因子 (Peak Factor)
**作用**：峰值因子用于表示信号中最大的峰值幅度，反映了信号的最大冲击力。它在机械故障诊断中有重要应用，特别是用于检测高幅值的冲击事件。机械设备在运行过程中，若发生故障，如轴承或齿轮的破损，通常会产生瞬时的高幅值振动，这些振动会显著增加峰值因子。因此，通过监测峰值因子，可以快速识别出设备的异常状态，尤其是那些可能导致严重损坏的瞬时冲击事件。相比于其他振动特征，峰值因子更加直观地反映了信号的最大冲击力，便于及时发现和处理机械故障。

**计算公式**：

\[ \text{Peak Factor} = \frac{\text{Peak Value}}{\text{Mean Value}} \]

其中，\(\text{Mean Value}\)为信号的平均值。

### 4. 冲击脉冲指数 (Impact Pulse Index)
**作用**：冲击脉冲指数反映信号中冲击脉冲的频率和强度，是评估机械故障的重要参数。机械设备的故障，如轴承或齿轮的破损，通常会产生频繁而强烈的冲击脉冲，这些脉冲会显著增加冲击脉冲指数。因此，通过监测冲击脉冲指数，可以识别出设备的异常状态。高冲击脉冲指数表明信号中存在频繁和强烈的冲击脉冲，预示着机械故障的可能性。相比于其他振动特征，冲击脉冲指数更能反映出信号中冲击事件的频率和强度，便于及时发现和处理机械故障。计算方法通常通过时间域分析冲击脉冲的频率和幅值来进行。

## 调制特征 (Modulation Features)

调制特征用于描述信号中存在的调制现象，常用于齿轮箱故障诊断。

### 1. 幅度调制 (AM, Amplitude Modulation)
**现象描述**：幅度调制是指振动信号的幅值随时间变化，表现为包络的变化。机械系统中的幅度调制现象通常由机械松动、齿轮啮合故障或轴承损坏引起。这些故障会导致振动信号的幅值产生周期性或随机性的变化，反映在包络信号中。通过对振动信号进行包络分析，可以识别出幅度调制现象，从而评估机械系统的健康状态。幅度调制分析在故障诊断中具有重要意义，因为它能够识别出振动信号中的低频成分，这些成分通常与机械故障密切相关。通过包络检测算法获得包络信号，然后分析其变化特性，可以及早发现机械系统的异常情况，进行及时的维护和修理。

**计算方法**：通过包络检测算法获得包络信号，然后分析其变化特性。

### 2. 频率调制 (FM, Frequency Modulation)
**现象描述**：频率调制是指振动信号的瞬时频率随时间变化。机械系统中的频率调制现象通常由转速变化、齿轮箱故障或传动系统故障引起。这些故障会导致振动信号的瞬时频率产生周期性或随机性的变化，反映在信号的频率特性中。通过对振动信号进行频率分析，可以识别出频率调制现象，从而评估机械系统的健康状态。频率调制分析在故障诊断中具有重要意义，因为它能够识别出振动信号中的高频成分，这些成分通常与机械故障密切相关。通过希尔伯特变换等方法计算瞬时频率，可以及早发现机械系统的异常情况，进行及时的维护和修理。

**计算方法**：通过希尔伯特变换等方法计算瞬时频率。

### 3. 相位调制 (PM, Phase Modulation)
**现象描述**：相位调制是指信号的相位随时间变化。机械系统中的相位调制现象通常由周期性的机械故障引起。这些故障会导致振动信号的相位产生周期性或随机性的变化，反映在信号的相位特性中。通过对振动信号进行相位分析，可以识别出相位调制现象，从而评估机械系统的健康状态。相位调制分析在故障诊断中具有重要意义，因为它能够识别出振动信号中的相位变化，这些变化通常与机械故障密切相关。通过计算信号的相位并分析其变化特性，可以及早发现机械系统的异常情况，进行及时的维护和修理。

**计算方法**：通过计算信号的相位并分析其变化特性。

### 4. 调制谱 (Modulation Spectrum)
**现象描述**：调制谱是通过对调制信号进行傅里叶变换得到的频谱，显示了调制成分的频率和幅值。机械系统的周期性变化会反映在调制谱中。通过对调制谱的分析，可以识别出信号中的调制成分，从而评估机械系统的健康状态。调制谱分析在故障诊断中具有重要意义，因为它能够识别出振动信号中的周期性变化，这些变化通常与机械故障密切相关。通过傅里叶变换获得调制谱，可以及早发现机械系统的异常情况，进行及时的维护和修理。

**计算方法**：对包络信号或瞬时频率进行傅里叶变换。

## 谐波特征 (Harmonic Features)

谐波特征用于识别机械系统中谐波成分的存在，常见于电机、齿轮等系统中。

### 1. 谐波含量 (Harmonic Content)
**现象描述**：谐波含量分析信号的频谱，识别出基频及其谐波成分的幅值和频率。机械系统中的非线性效应或周期性故障会导致谐波出现。这些谐波成分在频谱中表现为基频及其整数倍的频率成分。通过对振动信号进行频谱分析，可以识别出谐波成分，从而评估机械系统的健康状态。谐波含量分析在故障诊断中具有重要意义，因为它能够识别出信号中的谐波成分，这些成分通常与机械故障密切相关。通过傅里叶变换获得频谱，识别并量化基频及其谐波成分的幅值，可以及早发现机械系统的异常情况，进行及时的维护和修理。

**计算方法**：通过傅里叶变换获得频谱，识别并量化基频及其谐波成分的幅值。

### 2. 总谐波失真 (THD, Total Harmonic Distortion)
**现象描述**：总谐波失真（THD）反映谐波成分相对于基频成分的总能量，用于评估信号的谐波失真程度。机械系统中的非线性失真通常是机械故障的标志。这些谐波失真成分在频谱中表现为基频及其整数倍的频率成分。通过对振动信号进行频谱分析，可以识别出谐波失真成分，从而评估机械系统的健康状态。总谐波失真分析在故障诊断中具有重要意义，因为它能够识别出信号中的谐波失真成分，这些成分通常与机械故障密切相关。通过傅里叶变换获得频谱，识别并量化谐波失真成分的幅值，可以及早发现机械系统的异常情况，进行及时的维护和修理。

**计算公式**：

\[ \text{THD} = \sqrt{\sum_{n=2}^{\infty} \left( \frac{V_n}{V_1} \right)^2} \]

其中，\(V_1\)为基频幅值，\(V_n\)为第\(n\)次谐波的幅值。

### 3. 谐波频率 (Harmonic Frequencies)
**现象描述**：谐波频率识别振动信号中的谐波频率。机械系统中的周期性运动或电机驱动会产生谐波频率。这些谐波成分在频谱中表现为基频及其整数倍的频率成分。通过对振动信号进行频谱分析，可以识别出谐波频率，从而评估机械系统的健康状态。谐波频率分析在故障诊断中具有重要意义，因为它能够识别出信号中的谐波频率，这些频率通常与机械故障密切相关。通过傅里叶变换获得频谱，识别并量化谐波频率，可以及早发现机械系统的异常情况，进行及时的维护和修理。

**计算方法**：通过傅里叶变换识别频谱中的谐波频率。

### 4. 谐波包络 (Harmonic Envelope)
**现象描述**：谐波包络反映谐波信号的包络变化。机械系统中的周期性故障会导致谐波包络变化显著。这些谐波成分在频谱中表现为基频及其整数倍的频率成分。通过对谐波信号进行包络分析，可以识别出谐波包络的变化，从而评估机械系统的健康状态。谐波包络分析在故障诊断中具有重要意义，因为它能够识别出信号中的谐波包络变化，这些变化通常与机械故障密切相关。通过包络检测算法分析谐波信号的包络，可以及早发现机械系统的异常情况，进行及时的维护和修理。

**计算方法**：通过包络检测算法分析谐波信号的包络。

## 声品质特征 (Acoustic Quality Features)

### 1. 响度 (Loudness)
**作用**：响度反映了声音的强度感知，是评价噪声的重要指标。响度是指人耳对声音强度的主观感受，通常与声音的物理强度有关。机械设备的故障通常伴随着响度的增加，因为故障会产生更大的振动和噪声。通过监测设备运行时的响度变化，可以识别出潜在的机械故障。响度的计算通常基于心理声学模型，如ISO 532 B标准，该标准考虑了人耳对不同频率和声压级的响应特性。定期监测机械设备的响度，可以帮助工程师及早发现故障，进行必要的维护，避免设备损坏和停机。

**计算方法**：通过心理声学模型计算，如ISO 532 B。

### 2. 粗糙度 (Roughness)
**作用**：粗糙度描述声音的刺耳程度，与调制频率和深度有关。粗糙度是指声音中快速振幅调制成分引起的刺耳感，通常由机械设备的振动不均匀性引起。机械设备的故障，如齿轮啮合不良或轴承磨损，会导致振动信号的不规则变化，增加粗糙度。通过监测振动信号的粗糙度变化，可以识别出设备的异常状态。粗糙度的计算依赖于心理声学参数和振动信号的调制特征。高粗糙度通常表明设备运行状态不佳，可能需要进行检查和维修。粗糙度是声品质分析中的重要指标，有助于提高机械设备的运行可靠性和安全性。

**计算方法**：依赖于心理声学参数和振动信号的调制特征。

### 3. 尖锐度 (Sharpness)
**作用**：尖锐度反映了声音的高频成分的感知程度。尖锐度是指人耳对声音中高频成分的敏感度，高频成分越多，声音越尖锐。机械设备的高频振动通常与故障相关，如齿轮啮合不良或轴承损坏会产生高频噪声。通过监测设备运行时的尖锐度变化，可以识别出潜在的机械故障。尖锐度的计算通常基于高频成分的能量分布，考虑了人耳对不同频率的响应特性。定期监测机械设备的尖锐度，可以帮助工程师及早发现故障，进行必要的维护，避免设备损坏和停机。

**计算方法**：通过高频成分的能量计算。

### 4. 声音品质因子 (Sound Quality Index)
**作用**：声音品质因子综合反映声音的总体品质，是评价机械设备运行状态的重要指标。声音品质因子考虑了声音的多个方面，包括响度、粗糙度和尖锐度等。机械设备运行状态良好时，声音品质因子较高，反之，设备故障时，声音品质因子会显著降低。通过监测设备运行时的声音品质因子变化，可以识别出潜在的机械故障。声音品质因子的计算通常基于多种心理声学参数的综合评估。定期监测机械设备的声音品质因子，可以帮助工程师及早发现故障，进行必要的维护，避免设备损坏和停机。

**计算方法**：通过综合评估多种心理声学参数计算。
